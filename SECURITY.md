# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in JANE AI, please
report it responsibly:

**Email:** ravinduvidulaka@gmail.com

**Please include:**
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Your suggested fix (if any)

**Please do NOT:**
- Open a public GitHub issue for security vulnerabilities
- Share the vulnerability publicly before it is fixed
- Attempt to exploit the vulnerability

## Security Features

JANE AI includes the following security measures:

- **Cryptographic Authentication** — Verifies creator identity
- **File Integrity Protection** — Detects unauthorized modification
- **Anti-Tampering Measures** — Detects reverse engineering attempts

## Privacy

All processing happens locally on your device.
No data is sent to any external server.
See the [LICENSE](LICENSE) for complete privacy details.
